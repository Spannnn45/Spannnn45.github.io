My projects

  • gpt-3 bot

    This is a discord bot written in python with discord.py. It uses the openai api to communicate with gpt-3. You can use it yourself by creating your own discord application and openai account. Its very fun to play with and you can also play music with it. I made it with hacktail on github and he wrote the gpt-3 part and i wrote most of the bot. This was made for mattekollo 2022 project presentation in about 5 days. Sad that some companys have started using openai for their own profit.

  • frmWrk

    frmWrk is a custom made webserver that only works with HTTP 1.1. It's made in python and only uses sockets to communicate with the client. Im not very proud of this but it was made in 10 days for mattekollo 2022 project presentation. It takes an object oriented approach and the user has to do a lot of the work but the webserver does some of the heavy lifting.

    Side note: this website uses frmWrk.

  • project-selector

    project-selector is a side project for learning rust. I got some help from my friend emcat but i did much of the work on my own. I still use it to quickly navigate to my project folders. It's the only one of my projects to use a config file and im pretty proud of it.

  • christmastree-leds

    christmastree-leds is a program for neopixel leds written in Arduino C++. It still includes an old version written a year ago for the previous christmas but the new version is the best. It includes four modes and you can set the hue of some of them.
